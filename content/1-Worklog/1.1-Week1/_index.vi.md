### Kết quả đạt được tuần 1:

* Hoàn thành việc tham gia chương trình First Cloud AI Journey và nắm được lộ trình học tập.

* Thiết lập thành công tài khoản AWS và kích hoạt gói AWS Credits trị giá 100 USD.

* Hiểu được các nhóm dịch vụ cơ bản của AWS gồm:
  * Compute
  * Storage
  * Networking
  * Database
  * Security

* Làm quen với AWS Management Console và biết cách truy cập, quản lý các dịch vụ trên giao diện web.

* Cài đặt và cấu hình thành công AWS CLI bao gồm:
  * Access Key
  * Secret Access Key
  * Default Region
  * Output Format

* Thực hiện được các thao tác cơ bản với AWS CLI như:
  * Kiểm tra thông tin tài khoản
  * Xem danh sách Regions
  * Kiểm tra cấu hình CLI
  * Liệt kê các dịch vụ AWS
  * Thực hiện các lệnh quản lý tài nguyên cơ bản

* Hoàn thành việc thiết lập môi trường thực hành và sẵn sàng cho các nội dung Cloud chuyên sâu ở những tuần tiếp theo.
