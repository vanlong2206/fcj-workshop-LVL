### Kết quả đạt được tuần 9:

* Hoàn thành toàn bộ **Module 6** trên nền tảng Amazon Web Services.
* Nắm được kiến thức về:
  * Amazon RDS và Amazon Aurora.
  * Amazon Redshift và Amazon ElastiCache.
  * Database Backup & Restore.
  * AWS Database Migration Service (DMS).
* Hoàn thành các bài Lab thuộc Module 6, bao gồm:
  * Triển khai Amazon RDS.
  * Cấu hình VPC, EC2 và DB Subnet Group.
  * Backup và Restore cơ sở dữ liệu.
  * Database Migration Service.
  * Schema Conversion.
  * Migration Task.
  * Troubleshooting quá trình di chuyển dữ liệu.
  * Resource Cleanup.
* Hoàn thành các nội dung đầu tiên của **Module 7**, bao gồm:
  * Amazon Kinesis Data Firehose.
  * AWS Glue.
  * Amazon Athena.
  * Amazon QuickSight.
  * Amazon DynamoDB.
* Thực hành xây dựng pipeline xử lý dữ liệu, phân tích dữ liệu bằng Athena và trực quan hóa dữ liệu với QuickSight; đồng thời làm quen với Amazon DynamoDB và các thao tác quản lý cơ sở dữ liệu NoSQL.
* Cập nhật đầy đủ kết quả học tập và thực hành vào nhật ký công việc, sẵn sàng tiếp tục các nội dung còn lại của Module 7 trong tuần tiếp theo.
